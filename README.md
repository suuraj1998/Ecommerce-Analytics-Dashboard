# E-Commerce Sales & Profitability Analytics Dashboard

## Project Overview

This project analyzes sales performance, profitability, customer behavior and returns using Excel, Power Pivot and DAX.

## Tools Used

- Microsoft Excel
- Power Pivot
- DAX
- Pivot Tables
- Pivot Charts
- Data Modeling

## Key Insights

- Electronics contributes highest revenue.
- East region generates highest profit.
- Higher discounts reduce profitability.
- Product returns impact overall margins.
- Customer concentration drives revenue.
